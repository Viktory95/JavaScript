Assignment problem - программа для решения задачи о назначениях.
Решение производится венгерским методом.
Необходимо задать размер матрицы, нажать кнопку "Ввести значений",
и перед пользователем появится матрица указанного размера для заполнения.
После необходимо нажать на кнопку "Решить", после чего программа выдаст ответ
и подсветит поля матрицы, выбранные для ответа.
Можно нажать кнопку "Очистить", если ошиблись с размером матрицы, но это не обязательно,
потому что по нажатию на кнопку "Ввести значений" программа очистит форму прежде чем вывести новую матрицу для пользователя.
Можно нажать кнопку "Взять данные из файла", предварительно указав размер матрицы, после чего программа сама выедет матрицу на форму
и заполнить ее значениями из файла.
